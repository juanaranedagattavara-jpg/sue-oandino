- **HTML**: Reducido de 2,539 a 630 líneas (75% menos código)


## 🚀 **Instalación y Uso**

1. **Clonar el repositorio**
```bash
git clone [repository-url]
cd sueño-andino
```

2. **Servir localmente**
```bash
# Python 3
python -m http.server 3000

# Node.js (si tienes http-server)
npx http-server -p 3000
```

3. **Abrir en navegador**
```
http://localhost:3000
```
